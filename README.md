# Library_chatbot
This project is a simple rule-based chatbot created using Python. The chatbot works like a basic college library assistant and can answer some common questions asked by students.

The chatbot uses if-else conditions and predefined intents to understand the user input and respond accordingly.

What the chatbot can do
Responds to greetings like hi, hello
Helps to check if a book is available in the library
Shows number of copies for a book
Gives basic idea about book prices
Tells library timings
Calculates fine for late return of books
Handles wrong input using exception handling
Fine Calculation

Fine is calculated as:

Fine = number of late days × 50

Error Handling

If user enters something wrong (like text instead of number), the chatbot shows an error message using try-except.

Tools Used
Python
Google Colab

This is a basic chatbot made for learning purpose. It can be improved further by adding more features and better responses.
